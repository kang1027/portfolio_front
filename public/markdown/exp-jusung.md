# 주성엔지니어링(주) R&D센터 정보기술팀

**기간**: 2020년 ~ 2021년
**소속**: [주성엔지니어링(주) R&D센터](https://www.jusung.com/document/domestic_branch) 정보기술팀
**역할**: ERP/PLM 관리자, 사내 자동화 도구 개발

## 회사 소개

주성엔지니어링은 반도체·디스플레이·태양광 장비를 자체 설계·생산하는 국내 대표 장비 기업입니다. R&D센터 정보기술팀은 사내 ERP/PLM 시스템 운영과 R&D 조직의 정보 인프라를 담당합니다.

## 주요 업무

### ERP / PLM 관리자

- 사내 ERP 및 PLM 시스템 운영 및 사용자 지원
- 권한/계정/마스터 데이터 관리
- 부서별 요청 대응 및 데이터 조회/보정

### 신규 그룹웨어 데이터 입력 자동화 프로그램 개발

- 그룹웨어 전환 시점에 발생한 대량 데이터 이관 자동화
- 반복적인 입력 업무를 줄이기 위한 사내용 자동화 도구 개발
- 데이터 정합성 검증 로직 포함

### 퇴사자 업무 처리 자동화 프로그램 개발

- 퇴사자 발생 시 ERP/PLM/그룹웨어 등 여러 시스템에서 수행해야 하는 권한 회수, 데이터 이관, 계정 정리 작업을 자동화
- 누락 가능성이 높은 휴먼 워크플로우를 코드로 표준화

## 배운 점

- 사내 시스템 관리자 관점에서 **데이터 정합성**과 **운영 자동화**의 중요성을 체감
- 사용자가 정해져 있는 사내 도구를 만들 때의 **요구사항 정의 → 빠른 배포 → 피드백** 사이클을 경험
- 비개발 부서와의 커뮤니케이션, 권한·보안 관점에서의 시스템 설계 감각을 익힘

이 경험은 이후 풀스택 개발자로서 운영/보안/데이터 관점을 함께 고려하는 습관의 출발점이 됐습니다.
