# (주)오파크 — 풀스택 개발자

**기간**: 2026년 ~ 현재 (재직중)
**소속**: (주)오파크 (스타트업)
**역할**: 풀스택 개발자

## 회사 소개

(주)오파크는 빠른 사이클로 제품을 만들어 시장에 검증해나가는 스타트업입니다. 작은 팀이 제품 전반을 책임지고 굴러가는 환경에서, 기획·구현·배포·운영의 경계를 넘나들며 일하고 있습니다.

## 역할

스타트업 환경에 맞춰 **프론트엔드 / 백엔드 / 인프라 / 배포 운영**을 폭넓게 담당하는 풀스택 포지션으로 합류했습니다.

### 주요 책임

- 제품 기능 단위의 프론트엔드/백엔드 동시 구현
- API 설계 및 데이터 모델링
- 배포 파이프라인 구성 및 운영
- 신기능 도입 시 기술 스택·아키텍처 의사결정 참여

## 사용 기술

복무 이전부터 운영해온 스택을 그대로 활용하면서, 회사 제품에 맞춰 빠르게 기술을 추가해나가고 있습니다.

- **Backend**: Rust (Rocket / Tokio 기반 비동기 서버)
- **Frontend**: React, React Native, Flutter
- **Infra**: 셀프 호스팅 / 컨테이너 기반 배포
- **Workflow**: Claude Code를 활용한 AI 페어 프로그래밍

## 지향하는 일하는 방식

- **작게 자주 배포**하고 빠르게 피드백을 받는 사이클을 선호
- **운영까지 책임지는 코드**: 만들고 끝이 아니라, 장애·지표·로그까지 함께 보는 풀스택
- **AI-Native 워크플로우**: AI를 단순 보조가 아닌 페어 프로그래머로 활용해 1인분 이상의 처리량을 내는 것을 지향

> 더 자세한 회사/팀 소개는 입사 후 정보 공개 범위에 맞춰 업데이트할 예정입니다.
