# About This Site

This site is inspired by macOS [Big Sur](https://www.apple.com/in/macos/big-sur/) and [Catalina](https://www.apple.com/bw/macos/catalina/), developed using [React](https://reactjs.org/), [Zustand](https://zustand-demo.pmnd.rs/) and [UnoCSS](https://uno.antfu.me/).

The original template is from [Renovamen/playground-macos](https://github.com/Renovamen/playground-macos), customized for my portfolio.

## Tech Stack

- **Frontend**: React, TypeScript, UnoCSS
- **State Management**: Zustand
- **Build Tool**: Vite
- **Hosting**: Self-hosted

## Features

- macOS-like UI/UX
- Interactive desktop environment
- Terminal emulator
- Bear note-taking app
- Safari browser
- VS Code editor
- And more...
